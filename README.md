# Pre-Entrega 1 - JAVASCRIPT

------------

### CONTIENE 

Se intenta generar un sistema de compras para heladería, en el mismo se solicita al usuario que ingrese diferentes datos que se irán agregando al pedido y se utilizan como variables para sumar la cuenta final.

Datos que se solicitan:
- Nombre 
- Cantidad de helado
- Cantidad de barquillos pudiendo ser 0.


Para su construcción se utilizan los conocimientos aprendidos en clase. Variables, constantes, funciones, IF, ELSE, WHILE, SWITCHCASE. 
Todo el proceso se muestra en consola y se van generando alerts y prompt.
